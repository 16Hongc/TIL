#### 백준 알고리즘 (10996) - 별 찍기 - 21

 https://www.acmicpc.net/problem/10996



```python
num = int(input())                    # 입력 값 int 저장

for i in range(1, 2*num+1) :          # 홀짝이 한 세트이므로 x2만큼 반복 
    if i%2 == 1 :                     # 2로 나눈 나머지가 1(홀수)
        print('* ' * ((num + 1)//2))
    if i%2 == 0 :                     # 2로 나눈 나머지가 0(짝수)
        print(' *' * (num//2))
```
