#### 백준 알고리즘 (2523) - 별 찍기 - 13

 https://www.acmicpc.net/status?from_problem=1&problem_id=10039



```python
num = int(input())             # 입력 숫자를 int로 변환
cnt = 0                        
for i in range(1, num+1) :     # cnt에 1씩 더하며 *출력
    cnt += 1
    print('*'*cnt)
for j in range(cnt) :          # cnt에 1씩 빼며 *출력
    cnt -= 1 
    print('*'*cnt)
```
